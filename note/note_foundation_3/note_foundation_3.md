# 操作符
## 操作符
### 定义
- 对于字面值常量和变量进行操作的符号，也叫运算符
## 表达式
### 定义
- 用操作符将字面值常量和变量连接起来的式子,叫做表达式
### 运算符分类
- 算术运算符
    ```java
        + - * / %
        //求余运算，结果正负只跟表达式左边的操作数的符号有关
        +
        //表达式中只要有字符串参与，得到的结果都会是字符串
        //字符串加其他任意类型数据，结果都是字符串
        //注意区分字符串和字符

        ++
    ```
